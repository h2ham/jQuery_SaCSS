カプセル化と衝突の回避

jQueryでは、"jQuery"という変数に機能をつける仕組みをとっている。"jQuery"という変数を毎回書くのは記述が長くなってしまうので、省略して書けるようになっている。それが"$"である。

jQuery === $

しかし他のライブラリでも"$"を機能として使っている場合がある。同時に指定された場合などで影響がでる可能性があるために、どんな場合でも問題なく利用できるために、カプセル化（隠蔽）しておく必要がある。

カプセル化

(function($){
  // Your code here
})(jQuery);

最初の($)を一番最後の(jQuery)として認識させます。

DOM Readyでの実行の場合

jQuery(function($){
  // Your code here
});

これは下記の省略形

jQuery(document).ready(function($){
  // Your code here
});

最初はよくわからないかもしれませんが、慣れないうちは、HTML中のどこで指定したとしても

jQuery(function($){
  // Your code here
});

これを最初に書いて、// Your code hereの部分に必要なコードを書いていくようにしましょう。
