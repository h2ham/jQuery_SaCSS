jQueryとは

簡単で簡潔な記述で様々な機能や効果を付けられるようになる、JavaScriptのライブラリです。

The Write Less, Do More, JavaScript Library（少ない記述でより多くのことを行うためのJavaScriptライブラリ）

JavaScriptとjQuery

JavaScript

divのclassがinnerの要素の横幅を10%に変更するJavaScriptを書くとき。

var elements =  document.getElementsByTagName('div');
for(var i = 0; i < elements.length; i++) {
	if(elements[i].className === 'inner') {
		elements[i].style.width = '10%';
	}
};

jQuery

JavaScriptよりも短く書ける。

$('div.inner').css('width','10%');

横幅のサイズ変更であればこんな書き方もできる。

$('div.inner').width('10%');

セレクタの指定も様々

$('div').filter('.inner').width('10%');
