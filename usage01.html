jQueryとは

簡単で簡潔な記述で様々な機能や効果を付けられるようになる、JavaScriptのライブラリです。

The Write Less, Do More, JavaScript Library
（少ない記述でより多くのことを行うためのJavaScriptライブラリ）

JavaScriptとjQuery

JavaScript

divのclassがinnerの要素の横幅を10%に変更するJavaScriptを書くとき。

var elements =  document.getElementsByTagName('div'); // 要素を取得して
for(var i = 0; i < elements.length; i++) { // 要素から抽出するためにループ
	if(elements[i].className === 'inner') { // class名がinnerを抽出
		elements[i].style.width = '10%'; // 横幅のスタイルを10%に
	}
};

jQuery

JavaScriptよりも短く書ける。

$('div.inner').css('width','10%');

横幅のサイズ変更であればこんな書き方もできる。

$('div.inner').width('10%');

セレクタの指定も様々

$('div').filter('.inner').width('10%');

jQueryを使うメリット

  • CSSと近い感覚でHTML要素の取得が可能になる
    ⇒ デザイナー・コーダーでも扱いやすい
  • クロスブラウザに対応した様々な機能をつけられる
    ⇒ ブラウザ毎の差異を気にしなくてもOK
  • 軽量かつ高速
